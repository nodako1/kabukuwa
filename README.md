# カブクワの夏休み

世界中のカブトムシ・クワガタムシだけを探す、田舎の夏休み採集ゲームです。

第一段階では、ブラウザで1日を最後まで遊べる縦切り版を実装しています。

## 第一段階で遊べること

- 6:00から18:00まで村のマップを探索
- 位置が固定された木やトラップを選び、虫の有無を確認
- 探索へ入り直すと虫がいる場所だけが変化
- カブトムシ・クワガタムシ5種を採集し、図鑑へ記録
- 5人のNPCと時間帯・日によって出会い、少しずつ進む会話を楽しむ
- 神社のおじさんとの3回の会話と16:00〜18:00を条件に秘密の森を発見
- 遠方で18:00になると、おばあちゃんの送迎イベント後に18:15・家へ帰還
- 18:15〜20:00は家と裏庭だけを探索
- 20:00に一日の採集結果を確認し、次の日へ進む
- 行動ごとの自動セーブと、選んだ木の前からの再開

## 広告報酬の扱い

実広告SDKにはまだ接続していません。差し替え可能な開発用モックで、次の3種類だけを実装しています。

- 出現率アップ
- 効果時間アップ
- おばあちゃんの追加ヒント

個体サイズの乱数キーには広告効果を含めていません。さらに、出現率アップによって追加出現した個体は将来のランキング対象外として記録します。自然出現なら、広告中に見つけてもランキング対象のままです。

## 実装方針

- Vite + React + TypeScript
- ゲームルールはUIから分離した純粋なReducer
- `worldSeed + day + location + visit + spot` による再現可能な乱数
- Zodで検証するバージョン付きLocalStorageセーブとバックアップ
- エリア、虫、NPC、会話をデータ定義から追加できる構造
- 完成イラストはアセットIDを介して後から一括差し替え
